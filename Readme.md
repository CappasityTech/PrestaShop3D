# Development

## Build module

```sh
# 0.0.0 - version of module
./prepare.sh 0.0.0
```

## Setup local PrestaShop server

```sh
./development-server.sh -v 1.7.1.2
```

Option | Default | Description
--- | --- | ---
`-v` | `1.6.1.18` | Version of PrestaShop from docker hub
